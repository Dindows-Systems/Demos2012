Libraries

Don't you hate searching for hosted scripts and stylesheets?

Well, here you'll find URLs to common JS and CSS libraries.

How do I use these libs?

For css libraries:


<link rel="stylesheet" href="[THE URL]">

PS: Replace [THE URL] with the URL of the library you want to use.

For js libraries:


<script src="[THE URL]"></script>

PS: Replace [THE URL] with the URL of the library you want to use.

Solving problems

1. When you don't know what the problem is always check the console.

2. Try doin' a google search on your problem.

3. Try a search on MDN.

If you really don't know what the problem is

Make a Fiddly, click on the share button and place it on StackOverflow.

I hope that'll solve the problem!

Some neat tricks

1. Use cdnjs.com if you need external libs like jQuery, Twitter Bootstrap, ...

2. Hate the default Twitter Bootstrap look? Use Bootswatch. There you'll find some awesome themes that'll make Bootstrap look awesome.

By the way: We used Bootswatch Cosmo to make this demo look awesome!

3. Always keep in mind that Fiddly actually edits the <body> of the HTML document.

4. Your code isn't saved untill you press the Run-button!

5. Fiddly works fully offline thanks to HTML5 LocalStorage and Offline Cache but remind that your external scripts and styles aren't saved offline!
